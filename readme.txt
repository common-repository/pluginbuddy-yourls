Allows you to insert YOURLS short links on a post or a page from a remote YOURLS install.

== Description ==

<a href='http://yourls.org'>YOURLS</a> is an incredibly easy-to-use URL shortener.  If you have this shortener remotely installed, this plugin will allow you to generate and insert short URLs onto a post or page.

This plugin doesn't automatically generate short links, but allows you to insert short links at will from the post editor.

= Features =
* Setting panel for setting the YOURLS configuration
* Shortcode icon for easy insertion into a post or page
* Dynamically generates and inserts links onto a post or page

== Installation ==

This plugin requires WordPress 3.0 or higher and PHP 5 or higher.

1. Just unzip and upload the "pluginbuddy-yourls" folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
